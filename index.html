Pre-Workshop Setup
Install Node.js v18+
nodejs.org → LTS download
Install Claude Code (CLI or Desktop app)
npm install -g @anthropic-ai/claude-code  or  claude.ai/download
Sign in or set an API key
claude login (Pro/Max)  or  export ANTHROPIC_API_KEY=sk-ant-...
Clone the workshop repo
git clone https://github.com/drejom/claude-workshop.git
Verify it works
CLI: claude --version  ·  Desktop: open app → point at a folder
